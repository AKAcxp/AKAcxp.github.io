# 炫酷自我介绍网页

这是一个使用 HTML、CSS 和 JavaScript 构建的炫酷个人自我介绍网页模板。

## 特点

- 响应式设计，适配各种设备
- 炫酷的视觉效果与动画
- 打字机效果的标题动画
- 视差滚动效果
- 技能进度条动画
- 时间线展示经历
- 交互式联系卡片

## 如何使用

1. 克隆或下载这个仓库到你的本地电脑
2. 使用文本编辑器打开`index.html`文件
3. 自定义内容:
   - 修改个人介绍文本
   - 更新技能列表和进度条
   - 编辑工作经验和时间线
   - 更新联系方式
   - 根据需要修改颜色和样式（在`styles.css`中）
4. 如需添加个人照片，替换`.img-placeholder`元素，添加实际图片
5. 通过浏览器打开`index.html`预览效果

## 自定义提示

- 在 CSS 变量中修改主题颜色:
  ```css
  :root {
    --primary-color: #你的颜色;
    --secondary-color: #你的颜色;
    // ...其他颜色变量
  }
  ```
- 要更改字体，替换 Google Fonts 导入并更新`font-family`
- 添加你自己的社交媒体链接到联系卡片
- 根据需要添加更多部分

## 技术栈

- HTML5
- CSS3 (动画, Flexbox, Grid, 媒体查询)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- ScrollTrigger
